## Guidance for Completing Entries

### Item Description Should Include:
- quantity  
- serial number  
- hostname  
- MAC/IP address  
- physical attributes (if any)  

### Chain of Custody Should Record:
- every transfer (time + person)  
- any access  
- justification for access  
- evidence sealing process  

---

## Notes
- Evidence must be stored securely with restricted access.  
- Only authorised personnel may handle evidence.  
- All evidence should be labelled clearly and consistently.
