# Third-Party Risk — Evidence Templates

Includes contract, liability, SIEM monitoring, and third-party data-feed validation templates.

---

## 1. SIEM Data Feed Validation Template

**Problem Identified:**  
Third party failing to send full logs to SIEM — missing authentication & error logs.

**Validation Checklist:**
- Required logs defined?  
- Logs mapped to fields?  
- Onboarding document exists?  
- Sample log export received?  
- Daily log volume monitored?  
- Alerting tested?  

**Evidence:**
- Log samples  
- SIEM dashboard screenshots  
- Ticket confirming fix  

---

## 2. Contract Clause Evidence Template

Includes:
- Security obligations  
- Data protection responsibilities  
- Breach notification timelines  
- Audit rights  
- Indemnity clause  
- Liability for credential leaks  

---

## 3. Third-Party Remediation Summary Template

**Issue:** Missing SIEM alerts due to incomplete log ingestion  
**Root Cause:** API integration misconfigured  
**Action:** Updated parser, verified log types  
**Evidence:**  
- Sample alerts  
- Mapping file  
- Ticket reference
